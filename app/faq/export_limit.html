FAQ

Is there a limit on how many Gene Ontology annotations I can export from QuickGO?

You can export upto 50,000 GO annotations via the QuickGO browser from the GO annotations page using the ‘Export’ button.

If you would like to access a larger set of annotations, please download the most suitable file from the UniProt-GOA downloads page at www.ebi.ac.uk/GOA/downloads.
